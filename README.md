## About

This project is a personal AI chat app built to practice frontend/backend integration with AI services. It's great for learning and demonstrating full-stack development.

---

## Features

- Chat with an AI assistant using the OpenAI API
- User authentication
- Persistent chat history
- Responsive design for mobile and desktop
- Clean and modern UI

---

## Tech Stack

**Frontend:**

- React using Vite
- Tailwind CSS
- Shadcn
- Zod
- Zustand
- Highlight.js
- Marked-react

**Backend:**

- Node.js with Express
- Passport.js
- Drizzle ORM
- Neon database

**AI & APIs:**

- OpenAI API

**Hosting:**

- Railway.com
